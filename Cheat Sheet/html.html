HTML Elements (tags)
Opening Tag Description Closing Tag
<h1> - <h6> Text headings, h1 being the largest and h6 being the smallest </h1> - </h6>
<p> Paragraph text

 </p>
<a> Anchor element that stores a link, which can be accessed by clicking on the anchor element on the page </a>
<head> Appears at the top of an HTML document and contains metadata not visible but relevant to he web page </head>
<body> The visible portion of the HTML document </body>
<header> A banner or other large portion of the web page usually at the top, a header may contain the name of the organization that owns the website, the title of the page of whatever website you're on, and perhaps a logo image </header>
<nav> Navigation for the site, contains links that lead to different pages of a website </nav>
<main> The main content area for your web page, this is a semantic tag to increase readability of your HTML document </main>
<footer> Any further information at the bottom of the page that a user may want to know about the website or organization that owns it </footer>
