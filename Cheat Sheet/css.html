CSS Selectors
Selector Description
div, h1, p, header, etc. Any HTML element can be used as a CSS Selector, but whatever properties get used will be applied to any matching element regardless of ID or class. This is a very non-specific way to select part of a web page.
. The "." denotes a defined class within an HTML doc, to use this selector simply use the "." followed by whatever the name of the class is you are trying to select. This will apply the styling within to all html elements with the specified class. This is a level of specificity above using HTML elements as CSS selectors.
# The "#" denotes a defined ID within an HTML doc, to use this selector simply use the "#" followed by whatever the name of the ID is you are trying to select. This will only apply the styling within to the one HTML element with this ID. An element's ID should be unique and used only when selecting an element who's styling will not need to be repeated. This is the most specific way to select an element from an HTML doc.
Common CSS Properties
Property Description
color Sets the color of text.
background-color Sets the background color of an element.
font-size Sets the size of the font.
margin Sets the outer margin of an element.
padding Sets the inner padding of an element.
border Sets the border around an element.
display Specifies the display behavior of an element.
position Specifies the positioning method of an element.
flex Defines the ability for a flex item to grow or shrink.
grid Defines a grid layout for an element.
opacity Sets the transparency level of an element.
overflow Specifies what happens if content overflows an element's box.
text-align Specifies the horizontal alignment of text.
vertical-align Specifies the vertical alignment of an inline or table-cell box.
width Sets the width of an element.
height Sets the height of an element.
Display Property Attributes
Attribute Description
flex Changes an element to a block-level container with flex items inside of it. May also be used to define flex-grow, flex-shrink, and flex-basis all in one declaration.
inline-flex Allows multiple flex containers to appear inline with each other.
block Defines an element as block-level, meaning it will occupy the full width available in its container. This is the default property for <div>, <p>, and <h1> - <h6>. Items displayed in this way will stack vertically on the page.
inline Displays an element as an inline element.
inline-block Displays an element as an inline-level block container.
grid Displays an element as a grid container.
none Removes the element from the document flow.
Flexbox
CSS Property Description Attribute Example
justify-content Used to space items along the main axis. flex-start, center, space-between
align-items Used to space items along the cross axis. stretch, flex-start, baseline
flex-grow Used to specify how much space (and in what proportions) flex items absorb along the main axis. 0, 1, 2
flex-shrink Used to specify how much flex items shrink and in what proportions along the main axis. 0, 1, 2
flex-basis Used to specify the initial size of an element styled with flex-grow and/or flex-shrink. auto, 100px, 50%
flex-wrap Specifies that elements should shift along the cross axis if the flex container is not large enough. nowrap, wrap, wrap-reverse
align-content Aligns flex lines within the flex container. flex-start, space-around, stretch
flex-direction Defines the direction of the flex items in the flex container. row, column, row-reverse
flex-flow Specifies flex-wrap and flex-direction in the same declaration. row nowrap, column wrap

CSS Grid

Each CSS Grid is made up of a parent container and child elements. These child elements can be styled independently of the grid container, and will stay in their specified location on the grid. The table below shows how to access and use CSS Grid containers.

Properties & Attributes
