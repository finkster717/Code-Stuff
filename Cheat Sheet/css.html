CSS Selectors
Selector Description
div, h1, p, header, etc. Any HTML element can be used as a CSS Selector, but whatever properties get used will be applied to any matching element regardless of ID or class. This is a very non-specific way to select part of a web page.
. The "." denotes a defined class within an HTML doc, to use this selector simply use the "." followed by whatever the name of the class is you are trying to select. This will apply the styling within to all html elements with the specified class. This is a level of specificity above using HTML elements as CSS selectors.
# The "#" denotes a defined ID within an HTML doc, to use this selector simply use the "#" followed by whatever the name of the ID is you are trying to select. This will only apply the styling within to the one HTML element with this ID. An element's ID should be unique and used only when selecting an element who's styling will not need to be repeated. This is the most specific way to select an element from an HTML doc.
